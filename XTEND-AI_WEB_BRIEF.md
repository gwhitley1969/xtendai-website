# Xtend‑AI Web Assets & Brand Notes (v1)

These assets were generated from the provided Xtend‑AI logo PNG.

## Included files
- `xtend-ai_logo_transparent_cropped.png` (full logo, transparent background)
- `xtend-ai_mark_transparent.png` (the X mark only, transparent background)
- `xtend-ai_wordmark_transparent.png` (wordmark only, transparent background)
- `favicon.ico` + `xtend-ai_favicon_*.png` sizes
- `xtend-ai_brand_tokens.css` (CSS variables + gradients)

## Brand colors (hex)
- Navy (deep): #00052A
- Navy (primary): #022A56
- Accent blue: #188CFF
- Link blue (AA on white): #0B6FE6
- Soft tint: #98B7D2

## Accessibility
- Avoid using #188CFF for small body text on white (contrast ~3.37:1).
- Use #0B6FE6 (or darker) for links and small interactive text on white.

## Logo usage
- Prefer SVG if available. Use PNG assets as fallback.
- Keep generous whitespace around the logo.
- For favicons/app icons, use the X mark only.
